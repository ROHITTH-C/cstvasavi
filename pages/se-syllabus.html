Software Engineering (SE) - Syllabus

Course Outline

Unit I: Introduction

Evolution, Software development projects, Exploratory style of software developments, Emergence of software engineering, Notable changes in software development practices, Computer system engineering. Software Life Cycle Models: Basic concepts, Waterfall model and its extensions, Rapid application development, Agile development model, Spiral model.

Unit II: Software Project Management

Software project management complexities, Responsibilities of a software project manager, Metrics for project size estimation, Project estimation techniques, Empirical Estimation techniques, COCOMO, Halsteadâ€˜s software science, risk management. Requirements Analysis And Specification: Requirements gathering and analysis, Software Requirements Specification (SRS), Formal system specification, Axiomatic specification, Algebraic specification, Executable specification and 4GL.

Unit III: Software Design

Overview of the design process, Characteristics of a good software design, Layered arrangement of modules, Cohesion and Coupling. Agility: Agility and the Cost of Change, Agile Process, Extreme Programming (XP), Other Agile Process Models, Tool Set for the Agile Process. Function-Oriented Software Design: SA/SD methodology, Structured analysis, Developing the DFD model of a system, Structured design, Detailed design, and Design Review. User Interface Design: Characteristics of a good user interface, Types of user interfaces, Fundamentals of component-based GUI development, and user interface design methodology.

Unit IV: Coding And Testing

Coding, Code review, Software documentation, Testing, Black-box testing, White-Box testing, Debugging, Program analysis tools, Integration testing, Testing object-oriented programs, Smoke testing. Software Reliability and Quality Management: Software reliability, Statistical testing, Software quality, Software quality management system, ISO 9000, SEI Capability Maturity Model, Six Sigma.

Unit V: CASE, Maintenance, and Reuse

Computer-Aided Software Engineering (CASE): CASE environment, CASE support in the software life cycle, Characteristics of CASE tools, Second generation CASE tools, CASE Architecture. Software Maintenance: Characteristics, Reverse engineering, Process models, Estimation of maintenance cost. Software Reuse: Definition, Introduction, Challenges in reuse, Basic issues, Reuse approach, Reuse at the organizational level.
