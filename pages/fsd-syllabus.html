Full Stack Development (FSD) - Exercises

Exercises

  • 1. Lists, Links, and Images
    • Develop an HTML program to demonstrate ordered, unordered, and nested lists.
    • Use the <a> tag to create hyperlinks with href and target attributes.
    • Create a webpage displaying an image gallery using thumbnails linked to full-size images.
  • 2. HTML Tables, Forms, and Frames
    • Design an HTML table using <table>, <tr>, <th>, <td> with attributes like border, rowspan, and colspan.
    • Create a timetable using a table with <caption>, cell spacing, and padding.
    • Develop a registration form using text fields, password fields, checkboxes, radio buttons, and dropdown lists.
    • Implement a webpage divided into three frames: an image, a paragraph, and a hyperlink.
  • 3. HTML5 and CSS Basics
    • Use HTML5 semantic tags like <article>, <section>, <aside>, <header>, <footer>.
    • Embed audio and video into a webpage.
    • Apply inline, internal, and external CSS styles to elements.
  • 4. CSS Selectors
    • Implement different types of selectors: simple, combinator, pseudo-class, pseudo-element, and attribute selectors.
  • 5. CSS Styling and Box Model
    • Demonstrate different ways to reference colors in CSS.
    • Apply background images with positioning and scrolling effects.
    • Use CSS font and text properties such as font-size, font-weight, text-decoration, and text-alignment.
    • Explain the CSS Box Model including content, border, margin, and padding.
  • 6. Applying JavaScript
    • Embed JavaScript internally and externally in a webpage.
    • Demonstrate different ways to display output.
    • Develop a prompt dialog box to ask for a voter's name and age, then display the eligibility result in a table.
  • 7. JavaScript Pre-defined and User-defined Objects
    • Use document, window, array, math, string, regex, and date object properties and methods.
    • Explain user-defined objects with properties, methods, accessors, and constructors.
  • 8. JavaScript Conditional Statements and Loops
    • Develop a program to find the largest number among three inputs.
    • Display weekdays using a switch case.
    • Print numbers using for, while, and do-while loops.
    • Print data in an object using for-in, for-each, and for-of loops.
    • Determine whether a number is an Armstrong number.
    • Calculate the denomination breakdown of a deposited amount.
  • 9. JavaScript Functions and Events
    • Develop functions to calculate factorial, Fibonacci series, prime numbers, and palindrome checks.
    • Create a webpage with buttons to execute the above functions.
    • Validate form inputs for name, mobile number, and email format.
